
## Module 10

| Command       | Explanation          |
| ------------- | -------------------- |
| `no cpd run`    | disable CDP globally |
| `cdp run`       | enable CDP globally  |
| `no cdp enable` | disable CDP on port  |
| `cdp enable`    | enable CDP on port                     |
